# Enterprise Employee Onboarding Platform

## Project Overview

This project demonstrates an enterprise-grade employee onboarding solution built using Microsoft Power Platform technologies.

The solution automates the onboarding process from employee request submission through approval, provisioning, training assignment, and reporting.

## Technologies Used

- Power Apps
- Power Automate
- SharePoint Online
- Microsoft Teams
- Outlook
- Dataverse
- Power BI
- Azure Active Directory

## Business Problem

Manual onboarding processes often create delays, missing approvals, inconsistent employee experiences, and security risks.

This solution automates onboarding and provides a centralized experience for HR, IT, managers, and employees.

## Features

- Employee onboarding requests
- Multi-level approval workflow
- Automated email notifications
- Teams notifications
- Role-based access control
- Equipment assignment tracking
- Training assignment automation
- Real-time Power BI dashboards

## Architecture

Power Apps → Power Automate → Dataverse/SharePoint → Teams/Outlook → Power BI
