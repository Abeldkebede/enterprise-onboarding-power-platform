# Solution Diagrams

This folder contains architecture diagrams and workflow diagrams for the Enterprise Employee Onboarding Platform.

## Diagrams

### High Level Architecture
Power Apps → Dataverse → Power Automate → SharePoint → Power BI

### Employee Onboarding Workflow
Employee Request → Manager Approval → HR Approval → IT Provisioning → Training Assignment → Reporting

### Security Architecture
Entra ID → Authentication → Role-Based Security → Dataverse Access

### Reporting Architecture
Dataverse + SharePoint → Power BI Dashboards
