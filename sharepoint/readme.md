# SharePoint

SharePoint Online serves as the document management and collaboration platform for the Employee Onboarding solution.

## SharePoint Sites

### HR Operations Site
- Employee onboarding documents
- Policies and procedures
- New hire forms

### IT Operations Site
- Equipment provisioning requests
- Access management requests
- Software deployment documentation

## SharePoint Lists

### Employee Requests
- Request ID
- Employee Name
- Department
- Status
- Submission Date

### Equipment Tracking
- Asset ID
- Device Type
- Assigned Employee
- Status

### Training Tracker
- Training Name
- Employee
- Due Date
- Completion Status

## Security Model

### HR Team
- Full Control

### Managers
- Contribute

### Employees
- Read Access

## Integrations

- Power Apps
- Power Automate
- Microsoft Teams
- Outlook
- Dataverse
