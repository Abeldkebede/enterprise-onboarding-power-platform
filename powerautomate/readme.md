# Power Automate

Approval workflows, notifications, provisioning, and business automation processes.

## Planned Flows

### Employee Onboarding Flow
- Employee submits onboarding request
- Manager receives approval request
- HR receives notification
- IT provisioning begins automatically

### Manager Approval Flow
- Approval routing
- Reminder notifications
- Escalation process

### Teams Notification Flow
- Send notifications to hiring managers
- Notify HR and IT teams

### Outlook Welcome Email Flow
- Welcome email automation
- Employee onboarding instructions

### Reporting Refresh Flow
- Refresh Power BI datasets
- Update onboarding dashboards
