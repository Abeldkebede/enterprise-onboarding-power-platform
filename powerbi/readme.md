# Power BI

Power BI dashboards provide executive reporting and onboarding analytics.

## Dashboard 1: Executive Overview

### KPIs
- New Hires This Month
- Onboarding Completion Rate
- Average Onboarding Time
- Equipment Provisioning Time

## Dashboard 2: Department Analytics

### Metrics
- Employees by Department
- Training Completion Status
- Pending Approvals
- Open Equipment Requests

## Dashboard 3: HR Operations Dashboard

### Metrics
- Active Requests
- Completed Requests
- Delayed Requests
- Employee Satisfaction Scores

## Data Sources

- Dataverse
- SharePoint Lists
- Excel
- Microsoft Forms
