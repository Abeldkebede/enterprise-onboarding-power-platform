# Technical Design Document

## Technology Stack

- Power Apps
- Power Automate
- Dataverse
- SharePoint Online
- Microsoft Teams
- Power BI
- Microsoft Entra ID

## Architecture Overview

Power Apps serves as the user interface.

Dataverse stores business data.

Power Automate manages workflow automation.

SharePoint stores onboarding documents.

Power BI provides reporting and analytics.

## Security

- Role-based access control
- Entra ID authentication
- Audit logging
- Data loss prevention policies
