# User Stories

## HR Administrator

As an HR Administrator, I want to submit onboarding requests so that new employees can be processed efficiently.

### Acceptance Criteria
- Employee information is captured
- Department is selected
- Manager is assigned
- Request is submitted successfully

---

## Hiring Manager

As a Manager, I want to approve onboarding requests so that employees can receive system access.

### Acceptance Criteria
- Approval email is received
- Request details are visible
- Approval or rejection is tracked

---

## IT Administrator

As an IT Administrator, I want to receive provisioning requests so that laptops and accounts can be prepared.

### Acceptance Criteria
- Equipment requests are generated
- Access requests are generated
- Completion status is tracked
