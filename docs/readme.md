# Project Documentation

This folder contains all project documentation used throughout the software development lifecycle.

## Documents

### Business Requirements Document (BRD)
Business goals, stakeholders, scope, assumptions, and requirements.

### Functional Design Document (FDD)
Application functionality, workflows, business rules, and user stories.

### Technical Design Document (TDD)
Architecture, integrations, security model, and deployment strategy.

### Deployment Guide
Environment setup, release process, and migration instructions.

### Support Guide
Troubleshooting procedures and operational support documentation.
